# MANZIL PROJECTS - Saudi Real Estate Management System

A comprehensive web application for managing real estate projects across Saudi cities.

## Features

- Manage projects in Riyadh, Jeddah, and Dammam
- Track apartments, floors, and project groups
- Real-time statistics and reporting
- Browser-based data storage
- Export/import functionality
- Responsive design for all devices

## Data Management

- Data is automatically saved to your browser's local storage
- You can export your data as a JSON file for backup
- Import previously exported data to restore your information

## Technology

- Pure HTML, CSS, and JavaScript
- No backend required - works entirely in the browser
- Responsive design for mobile and desktop
